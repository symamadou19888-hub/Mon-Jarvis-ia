# Communication entre les agents IA

## Objectif

Définir le système de collaboration entre les agents de Jarvis.

Le Directeur IA coordonne les agents et organise leur travail.

---

## Rôle du Directeur IA

Le Directeur IA :

- reçoit la demande de l'utilisateur ;
- analyse l'objectif ;
- choisit les agents nécessaires ;
- distribue les missions ;
- récupère les résultats ;
- valide la qualité finale.

---

## Format d'une mission

Chaque mission envoyée à un agent doit contenir :

- Objectif :
- Contexte :
- Travail demandé :
- Résultat attendu :
- Contraintes :

---

## Réponse d'un agent

Chaque agent doit retourner :

- Analyse réalisée ;
- Travail effectué ;
- Résultat obtenu ;
- Problèmes rencontrés ;
- Recommandations.

---

## Collaboration entre agents

Exemple :

Utilisateur :
"Créer une application mobile"

Directeur IA :

1. Architecte IA :
Définit la structure du projet.

2. Designer IA :
Crée l'expérience utilisateur.

3. Développeur IA :
Construit la solution.

4. Testeur IA :
Vérifie le fonctionnement.

5. Business IA :
Analyse la possibilité de commercialisation.

---

## Règles de communication

- Les agents doivent rester concentrés sur leur rôle.
- Les informations importantes doivent être transmises au Directeur IA.
- Les agents doivent signaler leurs limites.
- Aucun agent ne doit remplacer le rôle d'un autre agent.

---

## Vision

Créer une équipe d'agents IA capable de collaborer comme une véritable équipe professionnelle.
