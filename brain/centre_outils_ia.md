# Centre des Outils IA de Jarvis

## Mission

Gérer la liste des outils disponibles pour Jarvis et déterminer leur utilisation selon les besoins.

---

## Catégories d'outils

## Intelligence artificielle

### ChatGPT

Utilisation :
- raisonnement ;
- rédaction ;
- analyse ;
- assistance générale.

### Claude

Utilisation :
- analyse de documents ;
- réflexion longue ;
- architecture.

### Gemini

Utilisation :
- recherche ;
- multimédia ;
- services Google.

### Hermes et modèles open source

Utilisation :
- expérimentation ;
- solutions locales ;
- autonomie.

---

---

## Règles de sélection

Jarvis doit :

- choisir l'outil adapté à la mission ;
- privilégier les solutions gratuites ou accessibles ;
- éviter les outils inutiles ;
- demander validation avant les actions importantes.

---

## Objectif final

Créer un système capable de sélectionner automatiquement les meilleurs outils pour réaliser chaque mission.
